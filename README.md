# Makine Öğrenmesi ile Ev Fiyat Tahmini Projesi

Bu proje, Makine Öğrenmesi dersi için hazırlanmıştır.

## Dosya İçeriği
- `house_price_prediction.ipynb`: Projenin ana Jupyter Notebook dosyası. Tüm kodlar ve açıklamalar buradadır.
- `train.csv` ve `test.csv`: Veri setleri.
- `requirements.txt`: Gerekli Python kütüphaneleri.

## Kurulum ve Çalıştırma

1. Gerekli kütüphaneleri yükleyin (Eğer otomatik kurulum yapılmadıysa):
   ```bash
   # Windows için
   python -m venv venv
   .\venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. Jupyter Notebook'u başlatın:
   ```bash
   # Sanal ortam aktifken
   jupyter notebook house_price_prediction.ipynb
   ```
   
   **Not:** Notebook'u açtığınızda "Kernel" menüsünden **"Python (House Price Prediction)"** seçeneğini seçtiğinizden emin olun.

3. Notebook içindeki hücreleri sırasıyla çalıştırın.

## Notlar
- Proje raporunu Word formatında hazırlamayı unutmayın.
- Kodlar `house_price_prediction.ipynb` dosyasında açıklamalı olarak yer almaktadır.
